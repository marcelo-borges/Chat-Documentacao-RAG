📑 CT-e - Conhecimento de Transporte Eletrônico
🔹 Objetivo
Definição: Documento fiscal emitido por transportadoras para cobrir o serviço de frete.

🔹 Regras de Negócio
Vínculo com NF-e
Regra: Um CT-e obrigatoriamente deve referenciar uma ou mais chaves de acesso de NF-e (Notas de Produto).
Impacto: O peso e valor total do frete devem ser condizentes com a carga transportada.

Tomador do Serviço
Regra: O sistema deve permitir selecionar quem pagará o frete (Remetente, Destinatário ou Terceiro).

🔹 Regras de Tela
Importação de XML
Função: Botão para importar o XML da NF-e do cliente para preenchimento automático dos dados da carga no CT-e.

Cálculo de Frete
Visual: Exibição detalhada de Frete Peso, Ad Valorem, GRIS e Taxas de Despacho.

🔹 FAQ
O que é o DACTE?
Resposta: É o Documento Auxiliar do Conhecimento de Transporte Eletrônico, que deve acompanhar o motorista durante toda a viagem.

Posso emitir CT-e para pessoa física?
Resposta: Sim, desde que informados os dados do CPF e endereço completo para o cálculo de tributação interestadual (DIFAL).
